Історія кафедри

Кафедра комп'ютерного моделювання процесів і систем входить до складу НТУ «ХПІ» і готує фахівців, здатних розв'язувати складні інженерні та наукові задачі за допомогою сучасних комп'ютерних технологій. Кафедра поєднує фундаментальну підготовку з практичними навичками розробки.

Напрями підготовки

  • Комп'ютерні науки
  • Прикладна математика
  • Веб-технології та frontend-розробка
  • Штучний інтелект та аналіз даних

Дисципліни, що викладаються

  1. Сучасні Web-технології
  2. Frontend-розробка
  3. Комп'ютерні мережі та розподілені обчислення
  4. Методи та парадигми мов програмування
Приклад навчального коду (натисніть, щоб розгорнути)

На курсі Frontend-розробка студенти створюють функції для обробки даних. Наприклад, функція приймає змінну name і повертає привітання: 

function greet(name) {
  return `Вітаємо, ${name}!`;
}

console.log(greet("студенте"));
